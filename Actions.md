1. Add item to cart
    - Method
      - POST
    - URL
      - /customer_carts/{customer_id}/items
    - Inputs: 
      - Array of item attributes
        - Product ID
        - Quantity
        - Product Customization
      - Customer ID
    - Responses:
      - Cart ID
      - Items in Cart ID
    - Expected Result:
      - Item is added to cart with specified customizations
2. Remove item from cart
    - Inputs:
      -  Item in Cart ID
      -  Cart ID
    - Responses:
      -  Success/Error
    - Expected Result:
      - Item is removed from cart
